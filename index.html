Project 5: Recon a Site - Reconnaissance Report

Executive Summary: This repository contains a detailed reconnaissance report conducted on the Certified Hacker website (http://certifiedhacker.com/). The objective was to assess potential vulnerabilities through both passive and active reconnaissance techniques. The findings reveal critical information that could be exploited by malicious entities, underscoring the necessity for robust security measures.

Reconnaissance Findings:

  1. Passive Reconnaissance:
    • BuildWith: Conducted detailed analysis of technologies and service providers utilized by the website. Identified potential vulnerabilities including exposed server details and software versions.
    • Wappalyzer: Cross-referenced BuildWith findings to uncover OS versions and security protocols. Identified weaknesses that could be exploited for malicious activities.
    • Censys: Provided real-time server location, operating system details, and open TCP protocols. Exposed sensitive information potentially exploitable by attackers.
  2. Active Reconnaissance:
    • NSLOOKUP: Revealed organizational communication systems and related DNS records. Highlighted potential entry points for unauthorized access if exploited.
    • DigLookups / DNSBrutalattack / DNSEnumeration: Discovered IPV4 addresses, subdomains, and mail exchange servers. Attempted brute-force attacks to gather further information, emphasizing vulnerabilities.
  3. Summary: The reconnaissance findings underscore significant vulnerabilities within the infrastructure and security protocols of the Certified Hacker website. While some defenses against brute-force attacks are in place, critical information such as IP addresses and server configurations remain accessible. Addressing these vulnerabilities promptly is essential to mitigate potential cyber-attacks and safeguard sensitive data.

Skills and Technology:

Tools Used: BuildWith, Wappalyzer, Censys, NSLOOKUP, DigLookups, DNSBrutalattack, DNSEnumeration

Techniques Used: Passive and Active Reconnaissance, Cross-referencing

Skills Demonstrated: Cybersecurity Analysis, Vulnerability Assessment, Penetration Testing, DNS Querying, OS and Protocol Identification, Report Writing and Documentation
